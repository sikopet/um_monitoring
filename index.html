Um Monitoring

The Um interface monitoring system for GSM/GSM-R

Download this project as a .zip file Download this project as a tar.gz file

Files

  1. bin: the .exe file of the software

  2. data: the collected data including .xml and .xlsx

  3. picture: the hardware and software of the system

  4. Manual: the user's manual for the software in Chinese

How to

Make sure .NET Compact Framework 2.0 or above is installed in the OS system.

For users

  1. run the .exe file of /bin/CellInfo.exe

  2. configure the serial port

  3. click the run button

  4. more detailed information can be find in Manual.pdf

For developers

  1. open the CellInfo.csproj in Visual Studio

  2. try to make changes to different modules written in C#
